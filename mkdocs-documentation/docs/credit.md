# Sources and Credits

## Core Technologies

The PEPPOL module is built on top of several key technologies and standards:

- **Perfex CRM**: The foundation CRM platform
- **UBL (Universal Business Language)**: International standard for electronic business documents
- **PEPPOL Standards**: European standard for electronic document exchange
- **PHP**: Core programming language
- **XML Processing**: For UBL document generation and parsing

## Third-Party Libraries and Components

- **UBL Generation Libraries**: For creating compliant UBL documents
- **XML Parser Libraries**: For processing incoming PEPPOL documents
- **HTTP Clients**: For provider API communications
- **Validation Libraries**: For PEPPOL identifier and document validation

## Standards and Specifications

- **PEPPOL AS4**: PEPPOL Access Point specifications
- **UBL 2.1**: Universal Business Language standard
- **EN 16931**: European standard for electronic invoicing
- **ISO 6523**: International standard for organization identification
- **OpenPEPPOL**: PEPPOL governance and standards authority

## Provider Integrations

The module includes support for various PEPPOL access point providers:

- **Ademico**: Built-in provider integration
- **Provider Framework**: Extensible architecture for custom providers
- **Standard APIs**: Support for common PEPPOL provider interfaces
